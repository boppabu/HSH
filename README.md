# HighScoreHunt Website

A modern, minimalist website project with styling inspired by doublepoint.com, featuring a clean purple color scheme and responsive design.

## Features

- **Modern Design**: Clean, minimalist aesthetic with purple accent colors
- **Responsive Layout**: Fully responsive design that works on desktop, tablet, and mobile
- **Smooth Animations**: Fade-in effects and smooth scrolling
- **Sticky Navigation**: Persistent navigation bar with scroll effects
- **Component Sections**: Hero, Features, Product, Download, and Footer sections

## Project Structure

```
HighScoreHunt/
├── index.html              # Main HTML file
├── assets/
│   ├── css/
│   │   └── styles.css      # All styling with purple theme
│   ├── js/
│   │   └── main.js         # Interactive features and animations
│   └── images/             # Placeholder for images
└── README.md               # This file
```

## Design Features

### Color Scheme
- Primary Purple: #7c3aed
- Purple Dark: #5b21b6
- Purple Light: #a78bfa
- Clean neutral backgrounds with subtle gradients

### Typography
- System font stack for optimal performance
- Clear hierarchy with large hero titles (56px)
- Readable body text with proper line-height

### Sections
1. **Navigation**: Sticky header with logo, links, and CTA button
2. **Hero**: Large title, subtitle, CTA buttons, and hero image
3. **Features**: Three-column grid showcasing key features
4. **Product**: Two-column layout with content and image
5. **Download**: App store buttons with gradient background
6. **Footer**: Company info, links, and social media icons

## Getting Started

Simply open `index.html` in your web browser to view the website.

## Customization

- Update colors in `assets/css/styles.css` (CSS variables in `:root`)
- Replace placeholder images in the HTML
- Modify text content in `index.html`
- Add your own logo and branding

Works in all modern browsers including:

- Firefox
- Safari
- Edge
